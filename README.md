# pcb-defect-detection

用 Ultralytics YOLO26 偵測 PCB 裸板六類瑕疵（HRIPCB / PKU-Market-PCB 資料集）的求職作品集專案。

> 🚧 開發中：完整結果表、benchmark 與線上 demo 連結將於 Phase 2 完成後補上。
> 執行藍圖與所有技術決策見 [plan.md](plan.md)。

## 六類瑕疵

`missing_hole`（漏鑽孔）、`mouse_bite`（鼠咬）、`open_circuit`（斷路）、`short`（短路）、`spur`（毛刺）、`spurious_copper`（多餘銅）
